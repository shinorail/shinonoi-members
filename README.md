
# 篠ノ井乗務区 Members Site（試作版）

このプロジェクトは、GitHub Pages 上で動作する  
**フォロワー限定・会員専用サイトの試作版**です。

## 機能一覧
- 共通ログイン（初期試作）
- 自動ID生成
- 個別アカウント登録
- ログイン保護（members.html）
- 403 / 404 / 500 エラーページ
- 企業サイト風デザイン

## 今後の予定
- X OAuth ログイン
- X API フォロワー判定
- Cloudflare Workers によるセキュリティ強化
- 会員ランク機能
- ギミック追加（アニメーション・鉄道風UI）
